# Lumina 使用说明

## 应用介绍

Lumina是一款功能丰富的智能笔记应用，集成了OCR文字识别、语音识别、图表绘制等功能，采用MVVM架构模式开发。本文档将帮助您快速了解和使用Lumina应用的各项功能。

## 系统要求

- Android 9.0+ (API Level 24+)
- 2GB RAM以上
- 100MB可用存储空间
- 网络连接（支持离线使用部分功能）

## 功能概述

### 基础功能
- **笔记管理**：创建、编辑、删除和归类笔记
- **富文本编辑**：支持文本格式化、列表、表格等
- **多媒体支持**：插入图片、音频、视频等
- **数据同步**：本地数据备份和云端同步

### AI增强功能
- **OCR文字识别**：从图片中提取文字
- **语音识别**：将语音内容转换为文字
- **智能标签**：自动生成笔记标签

### 用户体验
- **深色模式**：支持浅色/深色主题切换
- **多语言**：支持中文和英文界面
- **离线支持**：主要功能支持离线使用

## 基本操作

### 应用首页

1. **创建笔记**：
   - 点击右下角的"+"按钮
   - 选择笔记类型（文本/语音/图片）
   - 进入编辑界面

2. **笔记列表**：
   - 支持列表/网格视图切换
   - 长按笔记进行批量操作
   - 左滑/右滑快速操作

3. **搜索功能**：
   - 点击顶部搜索栏
   - 支持标题和内容搜索
   - 支持标签筛选

4. **菜单选项**：
   - 设置
   - 数据同步
   - 主题切换
   - 语言切换

### 笔记编辑

1. **文本编辑**：
   - 基本文本编辑
   - 格式化工具栏
   - 插入多媒体内容
   - 自动保存

2. **OCR识别**：
   - 点击工具栏OCR图标
   - 拍摄或选择图片
   - 确认识别结果
   - 编辑识别文本

3. **语音识别**：
   - 点击麦克风图标
   - 开始/停止录音
   - 查看识别结果
   - 编辑识别文本

4. **图表功能**：
   - 选择数据范围
   - 创建图表
   - 自定义样式
   - 更新数据
